## 1. Reverse Strings

Write program that:
•	Reads an input string
•	Reverses it using a Stack<T>
•	Prints the result back at the terminal

| Input  | Output |
| ------------- | ------------- |
| I Love C#  | #C evoL I  |
| Stacks and Queues  | seueuQ dna skcatS  |
